Цель работы:
Сделать 2D платформер, используя современную библиотеку для создания игр Pygame.

Системы игры:
	I. Текстуры и спрайты:
		1. Объединение всех тайлов в тайлсеты (tilesets);
		2. Возможность создать спрайт, размеры которого превышают размер обычного тайла;
		3. Разделение спрайтов по иерархии "динамические-статические объекты";
		4. Хранение текстур в папке textures на Яндекс.Диске;
	II. Карты:
		1. Хранение карт в формате .csv для сохранения и наглядности, и малого размера;
		2. Создание к каждой карте файла mapname_cdata.json для хранения информации о динамических объектах;
		3. Хранение карт в папке maps на Яндекс.Диске
	III. Звуки:
		1. Хранение аудиофайлов в формате .ogg по причине малого размера за счет повышенного сжатия;
		2. Произведение звуков в динамическом диапазоне для полного погружения в игровой процесс;
		3. Хранение звуков в папке sounds на Яндекс.Диске.
	IV. Сохранение:
		1. Реализация системы для хранения баланса, ключей, наигранного времени и прочего;
		2. Осуществление сохранения во время выхода из игры, при котором отображается диалоговое окно с подтверждением о закрытии сессии;
		3. Хранение файла сохранения игры в бинарном формате save.dat;
	V. Обновления:
		1. Введение версий игры для загрузки необходимых файлов при обновлении. Список загружаемых файлов описан в version_list.json;
		2. Запуск обновлений при стартовом окне, на котором также отображается кнопка старта и правила игры;
		3. Реализация системы обновлений, используя API Яндекс.Диска.

Активности в игре:
	I. Прохождение уровней. С возможностью загружать обновления для каждого игрового клиента, можно постоянно добавлять новые уровни-головоломки;
	II. Собирательство. В ходе игры можно собирать:
		1. Сундуки;
		2. Карточки;
		3. Деньги;
		4. Скейтборды;
